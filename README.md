# آفاق كلينك - AFAQ CLINIC

موقع عربي RTL حديث لعيادة آفاق كلينك في الرياض، مبني باستخدام Next.js App Router و TypeScript و Tailwind CSS و Framer Motion.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح:

```text
http://127.0.0.1:3000
```

## التخصيص

كل بيانات العيادة القابلة للتعديل موجودة في:

```text
lib/clinic-data.ts
```

يمكنك تعديل:

- أرقام الهاتف وواتساب
- العنوان وساعات العمل
- الخدمات والعلاجات
- العروض
- الأطباء المؤقتين
- الأسئلة الشائعة
- روابط التواصل الاجتماعي
- بيانات JSON-LD الخاصة بالعيادة

## ملاحظات مهمة

- لا توجد أسعار افتراضية داخل الموقع.
- أسماء الأطباء والمراجعات placeholders ويجب استبدالها ببيانات حقيقية موثقة.
- لا يتم تقديم وعود بنتائج مضمونة، وكل العبارات الطبية تستخدم صياغة آمنة مثل "يتم تحديد العلاج بعد الاستشارة الطبية".
- نموذج الحجز frontend-only حالياً ويمكن ربطه لاحقاً ببريد إلكتروني أو CRM أو نظام حجز.
